RetroCalculator
---------------

This is the RetroCalcluator iOS app. 

RetroCalclulator is a basic calculator app. The purpose of this project is to introduce and work with the concepts of:

* Math Operators
* UIStackView(s)
* IBActions
* IBOutlets
* Sounds
